# Run on Development

1. Clone repository
2. Run:
```
npm install
```
3. Instal Nest CLI:
```
npm i -g @nestjs/cli
```
4. Deploy mongo database on a docker container
```
docker-compose up -d
```

## Stack
* MongoDB
* NestJs
